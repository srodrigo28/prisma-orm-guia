Módulo 1: Referências e Requisitos

Antes de iniciar, valide o ambiente local e confirme a versão do Node.js.

Referências

  • https://www.nativewind.dev/
  • https://docs.expo.dev/
  • https://nodejs.org/en/download

Verificação de Node

🔎 Executar comando: verificar versao do Node
node -v

Se o comando falhar, instale o Node pelo link oficial e reinicie o terminal.

Regra: continuar com Node >= 20 (LTS recomendado).

Módulo 2: Criar Projeto Expo com TypeScript

Com o Node validado, crie seu app base em TypeScript.

Comandos

🆕 Executar comando: criar projeto Expo TypeScript
npx create-expo-app@latest app-nativewind --template blank-typescript
📂 Executar comando: entrar na pasta do projeto
cd app-nativewind

Módulo 3: Instalar NativeWind e Tailwind

Agora instale as dependências obrigatórias (incluindo Babel preset) e gere o arquivo base do Tailwind.

Instalação

🧩 Executar comando: instalar dependencias
npm install nativewind tailwindcss babel-preset-expo
🌐 Executar comando: instalar pacotes Expo/Web
npx expo install react-native-reanimated react-native-safe-area-context react-native-worklets react-dom react-native-web

Init Tailwind

📄 Executar comando: gerar arquivo base
npx tailwindcss init

Módulo 4: Configuração de Arquivos

Configure Tailwind, CSS global, Babel, tipagem e Metro para ativar o NativeWind.

Guia rapido antes de copiar os codigos (clique para expandir/recolher)

Siga esta sequencia para nao se perder:

  • 1️⃣ Criar arquivos novos: global.css, metro.config.js, babel.config.js, nativewind-env.d.ts (se ainda nao existirem).
  • 2️⃣ Substituir conteudo: tailwind.config.js e babel.config.js (se ja existirem).
  • 3️⃣ Nao alterar agora: index.ts, app.json, tsconfig.json.
  • 4️⃣ Salvar e validar: rode npx expo start -c para aplicar tudo com cache limpo.

✍️ Acao: substituir conteudo de tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./App.{js,jsx,ts,tsx}", "./src/**/*.{js,jsx,ts,tsx}"],
  presets: [require("nativewind/preset")],
  theme: { extend: {} },
  plugins: [],
};

🆕 Acao: criar arquivo nativewind-env.d.ts

/// <reference types="nativewind/types" />

🆕 Acao: criar arquivo global.css

@tailwind base;
@tailwind components;
@tailwind utilities;

✍️ Acao: substituir (ou criar) babel.config.js

module.exports = function (api) {
  api.cache(true);
  return {
    presets: [
      ["babel-preset-expo", { jsxImportSource: "nativewind" }],
      "nativewind/babel",
    ],
  };
};

🆕 Acao: criar arquivo metro.config.js

const { getDefaultConfig } = require("expo/metro-config");
const { withNativeWind } = require("nativewind/metro");
const config = getDefaultConfig(__dirname);
module.exports = withNativeWind(config, { input: "./global.css" });

Módulo 6: Execução e Troubleshooting

Rode o projeto, teste no dispositivo e limpe cache se necessário.

Erros comuns e correção

Erro: Cannot find module 'babel-preset-expo'

🛠️ Executar comando: corrigir preset Babel ausente
npm install -D babel-preset-expo npx expo start -c

Erro: .plugins is not a valid Plugin property

Confirme o babel.config.js exatamente como no Módulo 4 (com nativewind/babel em presets).

Teste limpo (do zero)

🧹 Executar comando: limpar e reinstalar
Remove-Item -Recurse -Force node_modules, package-lock.json, .expo, dist npm install npx expo start -c

Start do projeto

🚀 Executar comando: iniciar Expo
npx expo start

Atalhos: a Android, i iOS (macOS), ou QR Code no Expo Go.

Limpar cache

♻️ Executar comando: resetar cache
npx expo start -c

Validar dependências do SDK

🩺 Executar comando: diagnostico do projeto
npx expo-doctor

Se houver erro de peer dependency, rode npx expo install react-native-worklets.

Resultado esperado

Tela escura com card e textos estilizados por className, confirmando o NativeWind ativo.
