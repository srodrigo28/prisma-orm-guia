# Sequencia de Treinamento

Use os arquivos numerados nesta ordem:

1. `1-prisma-orm.html` - Setup completo do CRUD de produtos com Prisma
2. `2-guia-comandos.html` - Comandos essenciais do Prisma (consulta rapida)
3. `3-adicionando-categoria.html` - Inclusao da nova tabela `Categoria`
4. `4-relacionando-categoria-produto.html` - Relacao `1:N` entre `Categoria` e `Produto`

Observacao:
- Os arquivos sem numero foram mantidos para compatibilidade.
