# CRUD Prisma + Next.js

## Comandos principais

- Rodar o projeto: `npm run dev`
- Rodar o build: `npm run build`
- Aplicar migration: `npx prisma migrate dev --name <nome_da_migration>`
- Gerar Prisma Client: `npx prisma generate`
